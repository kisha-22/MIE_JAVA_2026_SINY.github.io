Queens of Streetwear

Queens of Streetwear is a full-stack C2C fashion e-commerce website that allows users to register, buy products, and list their own clothing items for sale. Built using HTML, CSS, JavaScript, PHP, and MySQL, the platform includes secure login and registration, seller dashboards, product management, and a dynamic shopping cart system. This project showcases my ability to develop a complete database-driven web application with user roles, authentication, and structured backend logic.

Technologies: HTML, CSS, JavaScript

Clarity Desk - Developer Mental Wellbeing Platform (Group Project)

Clarity Desk is a collaborative mobile application designed to support the mental wellbeing and productivity of remote developers through structured daily check-ins, mood tracking, and performance reflections. The platform enables users to monitor emotional trends over time using interactive dashboards and secure cloud-based data storage.

Developed using Flutter and Firebase, the application integrates user authentication, real-time database functionality, and responsive mobile UI design. As part of the development team, I contributed to feature implementation, Firebase integration, and the overall user experience structure.

Technologies: Flutter, Dart, Firebase, Cloud Firestore
